— The Problem

Why I built this

Managing a personal book collection across scattered notes, apps, and memory is messy. Bookshelf solves this by giving readers one clean place to store every book they've read — with ratings, personal reviews, and auto-fetched covers.

It's also a way to share reading recommendations with other students and book enthusiasts who can browse your shelf and discover their next read.

— Features

What it does

  • Full CRUD — Add, Edit, Delete books
  • ISBN validation via Open Library API (Axios)
  • Auto book cover fetch from Open Library Covers API
  • Star ratings + personal review notes
  • Sort library by newest or highest rated
  • Error handling for invalid ISBN and duplicates
  • Author–Book relational database design
  • Clean UI built with EJS partials
