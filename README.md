# Sreeman Dyageti — Portfolio

A bold, animated personal portfolio built from scratch with pure HTML, CSS, and JavaScript.

**Live Site:** [https://sreeman-dyageti.github.io/sreeman-portfolio/)

---

## Preview

> Full-stack developer portfolio with GSAP scroll animations, custom cursor, art showcase, and mobile-first responsive design.

---

## Built With

| Technology | Purpose |
|---|---|
| HTML5 | Structure |
| CSS3 | Styling & layout |
| JavaScript (Vanilla) | Interactivity |
| GSAP 3.12.5 | Animations & ScrollTrigger |
| Google Fonts | Bebas Neue + DM Sans |
| GitHub Pages | Hosting |

---

## Features

- **Hero section** — animated title lines with staggered GSAP entrance
- **Marquee ticker** — infinite scrolling tech stack strip
- **Work section** — asymmetric 12-column grid with 4 projects
- **Art section** — charcoal drawing showcase with Instagram-style lightbox
- **About section** — skills, spinning badge, bio
- **Contact section** — email link + social icons
- **Custom cursor** — GSAP-powered dot + ring that reacts to hover
- **Mobile nav** — fullscreen hamburger menu overlay
- **Scroll animations** — every section animates in and out on scroll
- **Fully responsive** — mobile, tablet, desktop

## Run Locally

No build tools needed. Just clone and open:

```bash
git clone https://github.com/sreeman-dyageti/portfolio.git
cd portfolio
# open index.html in your browser
```

Or use VS Code Live Server for hot reload.
